## Draft

[PIM Permission Sets Documentation]

Hello team :wave:

We know that PIM permission sets and access roles have not been properly documented until now, which has created confusion when teams need to understand what access they have or should request.

To address this, Process Engineering worked together with the IAM team (Carlos Marques) to create clear, centralized documentation that reflects the current as-is implementation in PIM.

**What's available now:**

The [Permission Sets in PIM](https://outsystemsrd.atlassian.net/wiki/spaces/RKB/pages/6192431127/Permission+Sets+in+PIM) page in the R&D Knowledge Base now provides:
- Complete list of permission sets currently implemented in PIM
- What each permission set grants access to
- Who should request each type of access

This documentation reflects what is live today. Any updates or changes to permission sets will be documented as they happen.

Thanks for your cooperation :gear:
