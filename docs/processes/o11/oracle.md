# Oracle

Here are a few &ldquo;did you know&rdquo; things that can be useful to you in the future (they were to me):

- 
Sequences in Oracle do not guarantee order by default. This can be changed but it impacts performance.

- 
We may ask for commit parameterization if we are unsure of data being timely updated: see how to ask this here.
