# ODC RPMs Priority Matrix

Problem prioritization considers the recurrence likelihood and impact of related incidents, which may refer to how serious the problem is from an infrastructure perspective (or service or customer perspective). 

**Low, Normal, High,&nbsp;**and&nbsp;**Urgent**&nbsp;are the 4 **Priorities **on the matrix that are automatically calculated in JIRA.
